SIM_MOM 系统功能说明及优化建议
============================

一、已实现功能
------------
1. 用户管理
   - 用户登录/登出
   - 用户列表（分页查询）
   - 新增用户
   - 编辑用户信息
   - 重置用户密码
   - 启用/禁用用户
   - 删除用户（单个/批量）

2. 个人中心
   - 查看/修改个人信息
   - 修改密码
   - 显示服务器时间

3. 安全特性
   - JWT token 认证
   - 密码加密存储
   - 密码强度验证
   - 操作日志记录

二、系统架构
----------
1. 前端技术栈
   - Vue 3
   - Element Plus
   - Vue Router
   - Axios

2. 后端技术栈
   - Spring Boot 3
   - MyBatis Plus
   - MySQL
   - JWT
