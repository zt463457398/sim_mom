> 生成时间: 2024-12-02 14:05:40

## 错误码设计规范
1. 错误码长度为5位数字
2. 第1位表示错误级别：1-系统级错误，2-业务级错误
3. 第2-3位表示模块：
   - 00: 系统通用
   - 01: 用户模块
   - 02: 认证模块
   - 03: 权限模块
   - 04: 文件模块
   - 05: 订单模块
   - 06: 产品模块
4. 第4-5位表示具体错误码
